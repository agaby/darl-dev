Connectivity

This determines how a model interacts with other remote services, and must be set up per model.

Azure Bot Services

You can connect this model to multiple bots in Azure. Each row coresponds to an external Bot reference in the ABS.

Obtaining these values is described in the tutorial

Seller center connection

In the near future our bots will support the Microsoft seller center connection as a way of monetizing rule sets. Enter your seller center data here.

Zendesk connection

If you have a zendesk account you can save feedback from the bots to it, by way of rulesets. see the report_bug.rule ruleset as an example of this. Set up you account data here.

Sendgrid connection

If you have a sendgrid account you can use it to send emails from your bots, by way of rulesets. see the call_andy.rule ruleset as an example of this. Set up you account data here.

Twilio connection

If you have a twilio account you can use it to send SMS texts from your bots, by way of rulesets. Set up you account data here.

Save connectivity

This will save the changes you made to connectivity to your model and to the routing mechanisms in DarlBot.
