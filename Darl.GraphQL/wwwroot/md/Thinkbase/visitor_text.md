# Thanks for visiting darl.dev and ThinkBase

This is Software as a Service that makes it possible to create Knowledge Graphs and re-use them in a variety of ways.

On this page you will see several samples that you can try out.

In each case you select a sample in the drop down. Some are really big and take a few seconds to load.

Once your sample is loaded you will see the __real view__.

This is the place where real world data items and relationships exist.

The __virtual view__ accessible through the _virtual_ tab collects metadata about  the real world. This is automatically generated, but you can place rules here.

The __recognition view__ contains the trees used to analyze user interactions with the Knowledge graph through the chat interface.

The __conversation view__ allows you to test the knowledge graph through a chat interface.

Try going to the __conversation view__ and start a conversation using the predefined text by clicking on the _arrow key_.

Everything about ThinkBase is designed to make creating Knowledge Graphs as easy as possible, but it's still a complicated subject.

We're happy to discuss any problems you might have with this subject on our [DiscordServer](https://discord.gg/4BaqGFukNG). Just ask a question.

To create your own Knowledge graphs click on __pricing__ and select a subscription.
