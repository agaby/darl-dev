# About Thinkbase
Thinkbase is a methodology for creating and using knowledge graphs (KGs). 
You develop KGs, in the 2D variant, through this page.

## Creating a new KG
Choose a unique name using only letters, numbers and the characters stop and underscore and enter it into the top right text box. Clicking Create will generate a new KG.
This will now appear in the Knowledge graphs dropdown. Select it to open it.

## Selecting an existing KG
Select a KG to open from the top left dropdown.

## Saving an edited KG
Save a KG with the __Save__ button. KG edits are not saved unless this is done.

## Copying an existing KG
You can create a copy of a KG using the __Copy__ button. You will be prompted for a new name.

## Deleting a KG
You can delete a KG by clicking on the __Delete__ button. This is irreversible.

# The editing tabs
## Real tab 
This is where you'll spend most of your time. It contains the Real World nodes, connections and attributes, typically corresponding to real things or actions or inferences on real things.
### Creating a node
Click in empty space and fill in the form that pops up
